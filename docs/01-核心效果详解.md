# 核心效果详解

## 一、背景层动效系统

### 1.1 AuroraBackground - 极光流体背景

**效果描述**
Canvas 绘制的流体噪声动画，模拟极光在夜空中流动的效果，融合三岛美学色彩。

**核心机制**
- **流体噪声算法**: 多层正弦/余弦函数叠加生成自然流动感
- **像素级渲染**: 直接操作 ImageData 实现高性能动画
- **CSS 光球叠加**: Framer Motion 驱动的模糊渐变球体

**技术实现**
```typescript
// 流体噪声函数 - 三层叠加创造自然流动
const noise = (x: number, y: number, t: number) => {
  return (
    Math.sin(x * 0.01 + t) * Math.cos(y * 0.01 + t * 0.5) * 0.5 +
    Math.sin(x * 0.02 - t * 0.3) * Math.sin(y * 0.015 + t * 0.7) * 0.3 +
    Math.cos(x * 0.005 + y * 0.008 + t * 0.2) * 0.2
  );
};
```

**色彩映射逻辑**
```typescript
// 金阁金 (#C9A227) 为基础，叠加噪声变化
const r = Math.floor(201 + n * 100 + n2 * 50);  // 红色通道: 201-351
const g = Math.floor(162 + n * 30 - n2 * 80);   // 绿色通道: 82-192
const b = Math.floor(39 + n2 * 100);            // 蓝色通道: 39-139
const a = Math.abs(n) * 0.15;                   // 透明度: 0-0.15
```

**CSS 光球层**
| 光球 | 尺寸 | 位置 | 动画周期 | 色彩 |
|------|------|------|----------|------|
| 主光球 | 800x800 | 左上 10%, 20% | 20s | 金→猩红渐变 |
| 次光球 | 600x600 | 右上 5%, 30% | 25s | 深海蓝→紫 |
| 三光球 | 500x500 | 中下 40%, 10% | 18s | 火焰橙→金 |

### 1.2 SakuraParticles - 樱花粒子效果

**效果描述**
50+ 樱花花瓣粒子从屏幕顶部飘落，随风摇摆旋转，营造物哀美学氛围。

**粒子属性**
```typescript
interface Particle {
  x: number;              // 水平位置
  y: number;              // 垂直位置 (从 -20 开始)
  size: number;           // 花瓣大小 2-6px
  speedX: number;         // 水平速度 -0.5~0.5
  speedY: number;         // 垂直下落速度 0.5~2
  rotation: number;       // 旋转角度 0-360°
  rotationSpeed: number;  // 旋转速度 ±0.025
  opacity: number;        // 透明度 0.3-0.8
  color: string;          // 花瓣色彩 (5种)
  swayAmplitude: number;  // 摇摆幅度 10-40px
  swayFrequency: number;  // 摇摆频率 0.01-0.03
  time: number;           // 时间偏移
}
```

**花瓣绘制算法**
```typescript
// 贝塞尔曲线绘制樱花花瓣形状
ctx.beginPath();
ctx.moveTo(0, -size);
ctx.bezierCurveTo(size * 0.5, -size * 0.5, size, 0, 0, size);
ctx.bezierCurveTo(-size, 0, -size * 0.5, -size * 0.5, 0, -size);
ctx.fill();

// 花瓣纹理线
ctx.moveTo(0, -size * 0.8);
ctx.lineTo(0, size * 0.8);
```

**色彩系统**
- `rgba(248, 244, 240, ?)` - 樱白
- `rgba(240, 230, 224, ?)` - 樱淡
- `rgba(255, 200, 200, ?)` - 浅粉
- `rgba(201, 162, 39, ?)` - 金阁金
- `rgba(255, 255, 255, ?)` - 纯白

### 1.3 CursorGlow - 鼠标跟随光晕

**效果描述**
双层弹簧物理延迟效果的光晕跟随鼠标移动，营造沉浸交互体验。

**物理参数**
```typescript
// 主光晕 - 快速跟随
const springConfig = { damping: 25, stiffness: 200 };
const cursorXSpring = useSpring(cursorX, springConfig);  // 响应快

// 次光晕 - 延迟跟随
const slowSpring = { damping: 30, stiffness: 100 };      // 响应慢
```

**三层结构**
| 层级 | 尺寸 | 模糊度 | 色彩 | 跟随速度 |
|------|------|--------|------|----------|
| 主光晕 | 256x256 | 40px | 金→猩红 | 快 (damping:25) |
| 次光晕 | 384x384 | 60px | 深海蓝 | 慢 (damping:30) |
| 核心光点 | 8x8 | 0px | 金阁金 | 即时 |

## 二、Hero 区域动效

### 2.1 逐字动画入场

**效果描述**
标题 "SantaChains" 每个字母依次入场，带模糊淡入效果。

**动画变体**
```typescript
const letterVariants = {
  hidden: { 
    opacity: 0, 
    y: 50, 
    filter: "blur(10px)" 
  },
  visible: (i: number) => ({
    opacity: 1,
    y: 0,
    filter: "blur(0px)",
    transition: {
      delay: i * 0.1,        // 每字延迟 0.1s
      duration: 0.8,
      ease: [0.215, 0.61, 0.355, 1],  // 贝塞尔缓动
    },
  }),
};
```

### 3.1 入场动画

**视口触发**

```typescript
const cardVariants = {

  },
  visible: {
    opacity: 1,

      delay: index * 0.15,    // 错开入场

    },
  },

```

### 3.2 悬停效果

**多层反馈**
```typescript
// 1. 背景光晕增强
<motion.div 
  animate={{ 
    opacity: isHovered ? 1 : 0.5, 
    scale: isHovered ? 1.05 : 1 
  }} 
/>

// 2. 边框颜色变化
className={`border-${isHovered ? '[#C9A227]/60' : '[#C9A227]/20'}`}

// 3. 引号装饰放大
<motion.span animate={{ scale: isHovered ? 1.1 : 1 }} />

// 4. 卡片上浮
whileHover={{ y: -8 }}

// 5. 发光阴影
className={isHovered ? glowMap[quote.color] : ""}
```

### 3.3 展开/收起动画

**AnimatePresence 实现**
```typescript
<AnimatePresence>
  {isExpanded && (
    <motion.div
      initial={{ height: 0, opacity: 0 }}
      animate={{ height: "auto", opacity: 1 }}
      exit={{ height: 0, opacity: 0 }}
      transition={{ duration: 0.4, ease: [0.215, 0.61, 0.355, 1] }}
    >
      {quote.context}
    </motion.div>
  )}
</AnimatePresence>
```
