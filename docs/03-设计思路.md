# 设计思路详解

## 一、美学理念

### 1.1 三岛由纪夫美学核心

**毁灭与美的辩证**
- 美在完成的瞬间达到顶峰，同时开始衰败
- 金阁寺的美在于它终将被烧毁的宿命
- 设计通过火焰动画、燃烧色彩体现这一张力

**肉体与精神的合一**
- 《太阳与铁》中的身体哲学
- 通过强烈的色彩对比（金/红/黑）表现肉体强度
- 动画的流畅性象征精神的自由

**瞬间与永恒的对立**
- 樱花飘落象征物哀（もののあはれ）
- 极光流动代表时间的绵延
- 语录卡片记录永恒的思考

## 二、动效设计原则

### 2.1 物理真实感

**弹簧动画系统**
```typescript
// 基于真实物理的弹簧参数
const springConfig = { 
  damping: 25,      // 阻尼系数 - 阻力大小
  stiffness: 200    // 刚度系数 - 弹性强度
};

// 应用：鼠标光晕跟随
cursorXSpring = useSpring(cursorX, springConfig);
```

**缓动函数选择**
| 场景 | 缓动函数 | 效果 |
|------|----------|------|
| 入场动画 | `[0.215, 0.61, 0.355, 1]` | 优雅减速 |
| 悬停反馈 | `easeOut` | 快速响应 |
| 背景流动 | `easeInOut` | 自然循环 |
| 粒子运动 | `linear` | 匀速飘落 |

### 2.2 视觉节奏

**时间配比**
- 入场动画: 0.8s (建立第一印象)
- 悬停反馈: 0.2-0.3s (即时响应)
- 背景循环: 18-25s (不干扰阅读)
- 语录轮播: 5s (留足阅读时间)

**延迟编排**
```typescript
// 逐字入场 - 每字延迟 0.1s
const letterVariants = {
  visible: (i: number) => ({
    transition: { delay: i * 0.1, duration: 0.8 }
  })
};

// 卡片入场 - 每张延迟 0.15s
const cardVariants = {
  visible: {
    transition: { delay: index * 0.15, duration: 0.8 }
  }
};
```

## 三、交互设计

### 3.1 悬停状态系统

**QuoteCard 悬停反馈**
```typescript
// 多层反馈叠加
onMouseEnter={() => setIsHovered(true)}

// 1. 背景光晕增强
<motion.div animate={{ opacity: isHovered ? 1 : 0.5, scale: isHovered ? 1.05 : 1 }} />

// 2. 边框颜色变化
className={`border-${isHovered ? '[#C9A227]/60' : '[#C9A227]/20'}`}

// 3. 引号装饰放大
<motion.span animate={{ scale: isHovered ? 1.1 : 1 }} />

// 4. 卡片上浮
whileHover={{ y: -8 }}

// 5. 发光阴影
className={isHovered ? glowMap[quote.color] : ""}
```

### 3.2 滚动触发机制

**视口检测**
```typescript
<motion.article
  initial="hidden"
  whileInView="visible"
  viewport={{ once: true, margin: "-100px" }}
/>
```

**参数说明**
- `once: true` - 只触发一次，不重复
- `margin: "-100px"` - 提前 100px 触发，流畅过渡

## 七、设计决策记录

### 7.1 为什么选择 Canvas 而非 CSS 动画？

| 方案 | 优点 | 缺点 | 选择 |
|------|------|------|------|
| Canvas | 像素级控制、流体效果 | 代码复杂 | ✅ 极光背景 |
| CSS | 性能好、简单 | 效果受限 | ✅ 光球动画 |
| WebGL | 3D 效果 | 过度设计 | ❌ 不适用 |

### 7.2 为什么选择 Framer Motion？

- 声明式 API，与 React 思维一致
- 内置弹簧物理系统
- 优秀的 TypeScript 支持
- 手势交互支持

### 7.3 为什么使用深色主题？

- 呼应能剧舞台的黑暗背景
- 突出金阁金等亮色元素
- 长时间阅读护眼
- 营造神秘、深邃的氛围
