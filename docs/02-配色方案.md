### 1.3 渐变组合方案

#### 主标题渐变
```css
background: linear-gradient(
  135deg, 
  #C9A227 0%,      /* 金阁金 - 起点 */
  #9B1B30 30%,     /* 猩红 - 转折 */
  #FF6B35 60%,     /* 火焰橙 - 高潮 */
  #C9A227 100%     /* 回到金色 - 循环 */
);
background-size: 200% 200%;
```

#### 光晕渐变
```css
/* 主光晕 - 金到猩红 */
radial-gradient(
  circle, 
  rgba(201, 162, 39, 0.3) 0%,    /* 中心金色 */
  rgba(155, 27, 48, 0.2) 40%,    /* 过渡猩红 */
  transparent 70%                /* 边缘透明 */
);

/* 次光晕 - 深海蓝 */
radial-gradient(
  circle, 
  rgba(26, 54, 93, 0.4) 0%,
  rgba(131, 56, 236, 0.2) 50%,
  transparent 70%
);
```

#### 边框渐变
```css
/* 装饰线渐变 */
linear-gradient(
  90deg, 
  transparent, 
  #C9A227,      /* 金色 */
  #9B1B30,      /* 猩红 */
  transparent
);

/* 滚动条渐变 */
linear-gradient(
  180deg, 
  var(--kin-kaku), 
  var(--blood-crimson)
);
```

### 1.5 透明度层级系统

| 层级 | 透明度 | 用途 |
|------|--------|------|
| 背景层 | 3% | Canvas 残影效果 |
| 玻璃态 | 3% | 卡片背景 (和纸质感) |
| 边框 | 10% | subtle 边框线 |
| 悬停边框 | 60% | 交互状态 |
| 文字主色 | 90% | 正文内容 |
| 文字次要 | 60% | 日期、标签 |
| 文字微弱 | 50% | 装饰元素 |

### 1.6 色彩使用规范

#### 文字色彩
```css
/* 主标题 */
color: #F8F4F0;                    /* 樱白 */
text-shadow: 0 0 80px rgba(201, 162, 39, 0.3);

/* 正文 */
color: rgba(248, 244, 240, 0.9);   /* 90% 樱白 */

/* 次要文字 */
color: rgba(201, 162, 39, 0.5);    /* 50% 金色 */

/* 日期 */
color: rgba(201, 162, 39, 0.5);
letter-spacing: 0.3em;
```

#### 背景色彩
```css
/* 主背景 */
background: #0a0a0a;               /* 墨黑 */

/* 卡片背景 */
background: linear-gradient(
  135deg, 
  rgba(248, 244, 240, 0.03) 0%,
  rgba(10, 10, 10, 0.5) 100%
);

/* 悬停光晕 */
background: radial-gradient(
  circle at center,
  rgba(201, 162, 39, 0.15) 0%,
  transparent 70%
);
```

#### 边框色彩
```css
/* 默认边框 */
border: 1px solid rgba(201, 162, 39, 0.1);

/* 悬停边框 */
border: 1px solid rgba(201, 162, 39, 0.3);

/* 装饰角标 */
border-left: 2px solid rgba(201, 162, 39, 0.2);
border-top: 2px solid rgba(201, 162, 39, 0.2);
```

### 3.2 应用示例

```tsx
// 根据主题动态应用样式
const QuoteCard = ({ quote }: { quote: Quote }) => {
  const theme = themes.find(t => t.id === quote.color);
  
  return (
    <div className={`bg-gradient-to-br ${theme?.gradient}`}>
      <h3 style={{ color: theme?.primary }}>{quote.content}</h3>
      <div style={{ borderColor: theme?.secondary }} />
    </div>
  );
};
```

## 四、对比度与可访问性

### 4.1 对比度检查

| 组合 | 前景 | 背景 | 对比度 | 等级 |
|------|------|------|--------|------|
| 正文 | #F8F4F0 | #0a0a0a | 15.2:1 | AAA |
| 金色文字 | #C9A227 | #0a0a0a | 7.8:1 | AA |
| 红色文字 | #9B1B30 | #0a0a0a | 5.2:1 | AA |
| 次要文字 | rgba(248,244,240,0.6) | #0a0a0a | 9.1:1 | AAA |

### 4.2 暗色模式优化

```css
/* 暗色主题默认 */
:root {
  --background: #0a0a0a;
  --foreground: #F8F4F0;
}

/* 高对比度模式 */
@media (prefers-contrast: high) {
  :root {
    --foreground: #ffffff;
    --kin-kaku: #FFD700;
  }
}

/* 减少动画 */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
