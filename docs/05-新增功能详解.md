# 新增功能详解 (v2.0)

本文档详细说明 v2.0 版本新增和优化的功能。

---

## 一、AuroraBackground - 极光背景增强

### 新增功能

#### 1. 鼠标交互影响
- 极光中心点随鼠标位置微动
- 光晕位置受鼠标影响产生视差效果

#### 2. 页面可见性检测
```typescript
const handleVisibilityChange = () => {
  isVisibleRef.current = document.visibilityState === "visible";
};
```
- 页面切换到后台时自动降低渲染频率
- 节省 CPU/GPU 资源

#### 3. GPU 加速优化
```typescript
<canvas
  className="will-change-transform"
  style={{ 
    filter: "blur(60px)",
    transform: "translateZ(0)",
  }}
/>
```
- 添加 `will-change-transform` 提示浏览器优化
- 使用 `translateZ(0)` 强制 GPU 渲染

#### 4. 响应式渲染优化
```typescript
const skipPixels = window.innerWidth < 768 ? 6 : 4;
```
- 移动端使用更大的像素步长（6px）
- 桌面端使用更精细的步长（4px）

## 二、SakuraParticles - 樱花粒子增强

### 新增功能

#### 1. 鼠标粒子交互
```typescript
// 粒子受鼠标吸引/排斥
const dx = mouseRef.current.x - particle.x;
const dy = mouseRef.current.y - particle.y;
const distance = Math.sqrt(dx * dx + dy * dy);

if (distance < maxDistance) {
  const force = (1 - distance / maxDistance) * 0.02;
  particle.speedX += dx * force * 0.1;
  particle.speedY += dy * force * 0.1;
}
```
- 鼠标靠近时粒子被吸引
- 离开时粒子恢复原始运动轨迹

#### 2. 自适应粒子数量
```typescript
const getParticleCount = () => {
  const width = window.innerWidth;
  if (width < 768) return 30;
  if (width < 1024) return 40;
  return 50;
};
```
- 移动端：30 个粒子
- 平板：40 个粒子
- 桌面：50 个粒子

#### 3. 性能优化
- 页面不可见时降低渲染频率（每 3 帧渲染 1 次）
- 使用 `devicePixelRatio` 优化高 DPI 屏幕

## 三、CursorGlow - 光标光晕增强

### 新增功能

#### 1. 点击波纹效果
```typescript
const addRipple = useCallback((x: number, y: number) => {
  const id = Date.now();
  setRipples((prev) => [...prev, { id, x, y }]);
  setTimeout(() => {
    setRipples((prev) => prev.filter((r) => r.id !== id));
  }, 1000);
}, []);
```
- 点击时产生金色波纹扩散效果
- 1 秒后自动消失

#### 2. 智能悬停检测
```typescript
const isInteractive = 
  target.tagName === "BUTTON" ||
  target.tagName === "A" ||
  !!target.closest("button") ||
  !!target.closest("a") ||
  target.classList.contains("cursor-pointer");
```
- 自动检测悬停在按钮、链接、可点击元素上
- 悬停时光晕放大 1.5 倍
- 核心光点缩小并增强发光

#### 3. 悬停状态指示器
- 悬停可交互元素时显示圆环指示器
- 提供视觉反馈

## 四、Hero - 首屏增强

### 新增功能

#### 1. 鼠标视差效果
```typescript
const handleMouseMove = (e: MouseEvent) => {
  const x = (e.clientX - rect.left - rect.width / 2) / rect.width;
  const y = (e.clientY - rect.top - rect.height / 2) / rect.height;
  setMousePosition({ x: x * 20, y: y * 20 });
};
```
- 中央光晕跟随鼠标偏移
- 每个字母有不同的视差深度
- 副标题轻微视差

#### 2. 平滑滚动动画
```typescript
const smoothOpacity = useSpring(opacity, { stiffness: 100, damping: 30 });
const smoothScale = useSpring(scale, { stiffness: 100, damping: 30 });
const smoothY = useSpring(y, { stiffness: 100, damping: 30 });
```
- 使用 `useSpring` 实现物理弹簧效果
- 滚动时更自然的淡出和缩放

#### 3. 增强入场动画
- 角落装饰依次入场（延迟 2s、2.1s、2.2s、2.3s）
- 装饰线添加发光效果
- 语录切换使用 AnimatePresence

## 五、QuoteCard - 语录卡片增强

### 新增功能

#### 1. 3D 倾斜效果
```typescript
const rotateX = useTransform(mouseYSpring, [-0.5, 0.5], ["8deg", "-8deg"]);
const rotateY = useTransform(mouseXSpring, [-0.5, 0.5], ["-8deg", "8deg"]);
```
- 卡片随鼠标位置 3D 倾斜
- 最大倾斜角度 ±8 度
- 使用弹簧物理效果

#### 2. 光泽扫过效果
```typescript
const glareX = useTransform(mouseXSpring, [-0.5, 0.5], ["0%", "100%"]);
const glareY = useTransform(mouseYSpring, [-0.5, 0.5], ["0%", "100%"]);

background: `radial-gradient(circle at ${glareX} ${glareY}, rgba(255,255,255,0.1) 0%, transparent 50%)`
```
- 鼠标移动时产生光泽扫过效果
- 模拟真实卡片反光

#### 3. 主题色边框
```typescript
const borderColorMap: Record<string, string> = {
  gold: "border-[#C9A227]/60",
  crimson: "border-[#9B1B30]/60",
  sea: "border-[#1a365d]/60",

```
- 悬停时边框变为对应主题色
- 角标颜色同步变化

#### 4. 3D 层级分离

- 不同元素有不同的 Z 轴深度
- 增强 3D 立体感

#### 5. 引号装饰动画
```typescript

  rotate: isHovered ? -5 : 0,

```
- 悬停时引号旋转、放大
- 增强交互反馈

## 六、性能优化总结

### 1. GPU 加速
- 所有动画元素添加 `will-change-transform`
- 使用 `translateZ(0)` 强制硬件加速

### 2. 节流策略
- 页面不可见时降低 Canvas 渲染频率
- 使用 `requestAnimationFrame` 确保流畅

### 3. 响应式优化
- 根据屏幕尺寸调整粒子数量
- 根据屏幕尺寸调整渲染精度

### 4. 内存管理
- 组件卸载时正确清理事件监听
- 取消动画帧请求
- 清理波纹效果定时器

## 七、交互设计原则

### 1. 物理真实感
- 使用弹簧物理参数（stiffness、damping）
- 缓动函数统一使用 `[0.215, 0.61, 0.355, 1]`

### 2. 视觉反馈
- 悬停状态明确指示
- 点击产生波纹反馈
- 过渡动画平滑自然

### 3. 性能优先
- 优先使用 transform 和 opacity
- 避免触发重排（layout）
- 后台时降低资源消耗
